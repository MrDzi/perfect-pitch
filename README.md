# 🎹 Check Your Pitch 🎹

## Pre-requisites

Node 16.x

## Getting started

- Clone the project

- Install the dependencies with the following commands:

```
yarn

cd client && yarn

cd ../server && yarn

cd ..
```

- Start the project

```
yarn dev
```

To build the client app, run:

```
yarn build:client
```

To build the server app, run:

```
yarn build:server
```
